# Toy Car Rental

`Toy Car Rental` — первый рабочий бизнес-плагин FIREBALL CMS для проката детских машинок.

## Назначение

Плагин позволяет оператору:

- выдавать машинку в прокат;
- задавать время поездки;
- видеть активные поездки;
- получать визуальное и звуковое напоминание о просрочке;
- завершать поездку;
- учитывать оплату;
- смотреть статистику за сегодня.

## Установка и активация

1. Откройте `Админка → Плагины`.
2. Найдите `Toy Car Rental`.
3. Нажмите `Установить`.
4. Нажмите `Активировать`.

После активации в меню админки появится пункт `Прокат машинок`.

## Добавление машинок

Откройте `Прокат машинок → Машинки → Добавить машинку`.

Для машинки задаются:

- название;
- номер;
- цвет;
- статус;
- цена за минуту;
- цена за поездку;
- изображение;
- порядок вывода.

## Запуск поездки

На странице `Панель оператора` свободная машинка показывает кнопку `Начать поездку`.

При запуске задаются:

- имя клиента;
- телефон;
- длительность 5 / 10 / 15 / 30 минут;
- сумма оплаты;
- способ оплаты;
- статус оплаты.

После старта создаётся запись в `toy_rental_rides`, а машинка получает статус `rented`.

## Таймер и просрочка

`assets/toy-rental.js` обновляет таймеры каждую секунду.

Когда время заканчивается:

- карточка подсвечивается красным;
- статус меняется на `Просрочена`;
- показывается toast;
- проигрывается короткий звук, если браузер разрешает аудио.

При загрузке страниц активные поездки с `planned_end_at < NOW()` серверно помечаются как `overdue`.

## Завершение поездки

Кнопка `Завершить`:

- записывает `ended_at`;
- считает фактическую длительность;
- ставит поездке `completed`;
- возвращает машинку в `available`;
- сохраняет оплату.

## История

Раздел `История` поддерживает фильтры:

- сегодня;
- вчера;
- период;
- машинка;
- способ оплаты;
- статус оплаты.

## Статистика

Раздел `Статистика` показывает:

- количество поездок сегодня;
- активные поездки;
- завершённые поездки;
- просроченные поездки;
- общую выручку;
- выручку наличными;
- выручку картой;
- среднюю длительность;
- популярную машинку.

## Настройки

Раздел `Настройки` хранит значения через `plugin_settings`:

- длительность по умолчанию;
- цена по умолчанию;
- валюта;
- звук;
- toast-уведомления;
- автообновление панели оператора.
