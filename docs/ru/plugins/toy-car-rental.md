# Toy Car Rental

`Toy Car Rental` — первый рабочий бизнес-плагин FIREBALL CMS для проката детских машинок.

## Назначение

Плагин позволяет оператору:

- выдавать машинку в прокат;
- запускать фиксированную или поминутную поездку;
- видеть активные поездки;
- получать визуальное и звуковое напоминание о просрочке;
- завершать поездку;
- учитывать оплату;
- смотреть статистику за сегодня.

## Установка и активация

1. Откройте `Админка → Плагины`.
2. Найдите `Toy Car Rental`.
3. Нажмите `Установить`.
4. Нажмите `Активировать`.

После активации в меню админки появится пункт `Прокат машинок`.

## Добавление машинок

Откройте `Прокат машинок → Машинки → Добавить машинку`.

Для машинки задаются:

- название;
- номер;
- цвет;
- статус;
- цена за минуту;
- цена за поездку;
- изображение;
- порядок вывода.

## Запуск поездки

На странице `Панель оператора` свободная машинка показывает кнопку `Начать поездку`.

При запуске задаются:

- имя клиента;
- телефон;
- тип аренды;
- для фиксированной поездки — длительность, сумма, способ и статус оплаты;
- для поминутной аренды — цена минуты и необязательное ориентировочное время.

После старта создаётся запись в `toy_rental_rides`, а машинка получает статус `rented`.

## Поминутная аренда

В поминутном режиме итоговая сумма на старте не вводится. Система считает время вверх от `started_at`, показывает примерную стоимость и сохраняет `payment_status = unpaid`.

При завершении сервер считает:

- фактические минуты с округлением вверх;
- `final_amount` как системно рассчитанную сумму;
- `payment_amount` как фактически оплаченную сумму, которую оператор может поправить вручную.

## Таймер и просрочка

`assets/toy-rental.js` обновляет таймеры каждую секунду.

Когда время заканчивается:

- карточка подсвечивается красным;
- статус меняется на `Просрочена`;
- проигрывается короткий звук, если браузер разрешает аудио.

При загрузке страниц активные поездки с `planned_end_at < NOW()` серверно помечаются как `overdue`.

## Завершение поездки

Кнопка `Завершить`:

- записывает `ended_at`;
- считает фактическую длительность;
- считает итоговую сумму для поминутной аренды;
- ставит поездке `completed`;
- возвращает машинку в `available`;
- сохраняет оплату.

## История

Раздел `История` поддерживает фильтры:

- сегодня;
- вчера;
- период;
- машинка;
- тип аренды;
- способ оплаты;
- статус оплаты.

## Статистика

Раздел `Статистика` показывает:

- количество поездок сегодня;
- фиксированные поездки;
- поминутные поездки;
- активные поездки;
- завершённые поездки;
- просроченные поездки;
- выручку только по оплаченным поездкам;
- выручку наличными;
- выручку картой;
- среднюю длительность;
- популярную машинку.

## Настройки

Раздел `Настройки` хранит значения через `plugin_settings`:

- длительность по умолчанию;
- цена фиксированной поездки по умолчанию;
- цена минуты по умолчанию;
- валюта;
- звук;
- автообновление панели оператора.

## Переводы

Все строки интерфейса плагина находятся в `plugins/toy-car-rental/lang/`. Плагин не добавляет свои переводы в основные файлы `app/Languages`.
