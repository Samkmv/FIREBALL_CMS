# Динамический контент

CMS отвечает за маршрутизацию, БД, страницы, записи, категории, поиск, настройки, SEO и пагинацию.

Тема отвечает только за HTML-представление переданных данных.

Правильно:

```php
<?php foreach ($posts as $post): ?>
    <h2><?= htmlSC($post['title']) ?></h2>
<?php endforeach; ?>
```

Неправильно:

```php
<?php // db()->query('SELECT * FROM posts'); ?>
```

Прямой SQL в теме ломает кеширование, обновления схемы, мультиязычность и Marketplace-проверки. Для дополнительных данных используйте Theme API.
