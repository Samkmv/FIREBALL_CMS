# Импорт и экспорт тем

FIREBALL CMS поддерживает перенос тем через ZIP-архивы.

## Экспорт

На странице тем нажмите `Экспорт`. CMS создаст ZIP-архив с файлами темы.

В архив попадают:

- `theme.json`
- `preview.png`
- `templates/`
- `partials/`
- `assets/`

Не попадают:

- `.DS_Store`
- `.env`
- `.htaccess`
- `.git/`
- `node_modules/`
- `vendor/`
- backup-файлы
- временные файлы

## Импорт

Откройте:

```text
Админка → Внешний вид → Импорт темы
```

Загрузите ZIP-файл. CMS распакует его во временную папку, проверит пакет и только потом перенесёт тему в `/themes/{slug}`.

## Проверки при импорте

CMS блокирует архив, если:

- нет `theme.json`;
- slug невалидный или зарезервирован;
- тема с таким slug уже существует;
- `minCmsVersion` выше версии CMS;
- есть пути с `../`;
- есть абсолютные пути;
- есть `.env` или `.htaccess`;
- есть PHP-файлы вне `templates/` и `partials/`;
- архив слишком большой;
- файлов слишком много.

## После импорта

CMS показывает информацию о теме:

- название;
- slug;
- автор;
- версия;
- описание;
- preview;
- кнопку активации;
- ссылку на список тем.

## Ошибки импорта

Если импорт не прошёл, временные файлы удаляются, активная тема не меняется, а пользователь видит причину ошибки.
