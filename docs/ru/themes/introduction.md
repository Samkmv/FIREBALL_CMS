# Темы FIREBALL CMS

Тема в FIREBALL CMS — это набор файлов, который отвечает за внешний вид публичного сайта. Тема содержит шаблоны страниц, частичные шаблоны, CSS, JavaScript, изображения и файл описания `theme.json`.

## Что делает тема

- Рендерит главную страницу, обычные страницы и записи.
- Подключает свои CSS и JS через helper `theme_asset()`.
- Хранит метаданные: название, slug, автор, версия, описание и минимальная версия CMS.
- Может быть активирована, экспортирована в ZIP и импортирована обратно.

## Что не должна делать тема

- Менять данные в базе напрямую.
- Подключать PHP-файлы вне `templates/` и `partials/`.
- Использовать пути с `../`.
- Хранить секреты, `.env`, токены, резервные копии и временные файлы.

## Активная тема и предпросмотр

Активная тема хранится в настройке `active_theme`. Предпросмотр открывает сайт с другой темой временно и не меняет эту настройку.

Пример предпросмотра:

```text
/?preview_theme=my_theme
```

Предпросмотр доступен только администратору.

## Рекомендуемый процесс

1. Создайте тему через админку.
2. Проверьте структуру файлов.
3. Откройте предпросмотр.
4. Исправьте шаблоны и assets.
5. Экспортируйте ZIP для переноса.
6. Активируйте тему только после проверки.
