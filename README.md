# FIREBALL CMS

Современная CMS на PHP с MVC-архитектурой, блочным редактором, системой пользователей, PWA-поддержкой, чатом и встроенным файловым менеджером.

---

## Возможности

* MVC архитектура
* PHP 8.0+
* Система пользователей и ролей
* Блочный редактор контента
* Публикация постов
* Категории
* AJAX интерфейсы
* Встроенный файловый менеджер
* Чат между пользователями
* PWA поддержка
* Service Worker
* SEO-friendly URL
* Система приоритетов публикаций
* Адаптивная админ-панель
* Загрузка изображений и медиа
* Composer autoload

---

## Установка

### 1. Клонирование проекта

```bash
git clone https://github.com/Samkmv/FIREBALL_CMS.git
```

---

### 2. Установка зависимостей

```bash
composer install
```

---

### 3. Настройка базы данных

Создай базу данных и импортируй SQL дамп.

Настройки подключения находятся в:

```txt
config/config.php
```

---

### 4. Настройка веб-сервера

Точка входа:

```txt
/public/index.php
```

Для Apache требуется mod_rewrite.
