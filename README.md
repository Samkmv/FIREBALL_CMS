# FIREBALL CMS

Современная CMS на PHP с MVC-архитектурой, мультиязычностью, системой тем, блочным редактором, встроенным установщиком и системой обновлений.

FIREBALL CMS создана как лёгкая, расширяемая и удобная система управления контентом для блогов, информационных порталов, каталогов, корпоративных сайтов и собственных веб-проектов.

---

* Страницы
* Записи (посты)
* Категории
* Теги
* Приоритеты публикаций

* Текст

## Установка

После загрузки проекта на сервер или запуска локально откройте сайт в браузере.

Если CMS ещё не установлена, автоматически откроется мастер установки:

```text
/install
```

Мастер установки позволит:

* Выбрать язык
* Настроить подключение к базе данных
* Указать название сайта
* Настроить URL сайта
* Выбрать часовой пояс
* Создать первого пользователя Creator

После завершения установки система автоматически:

* Создаст таблицы базы данных
* Создаст необходимые системные записи
* Создаст файл конфигурации
* Создаст первого пользователя Creator
* Защитит CMS от повторной установки

## Обновления

FIREBALL CMS поддерживает централизованное обновление через GitHub Releases.

Возможности:

* Проверка новых версий
* Обновление через административную панель
* Автоматический backup перед обновлением
* Защита пользовательских файлов
* Выполнение миграций после обновления

Во время обновления не затрагиваются:

* Пользовательские темы
* Загруженные файлы
* Конфигурация сайта
* Резервные копии

## Обслуживание базы данных

Для роли Creator доступен раздел обслуживания системы.

Функции:

* Очистка кэша
* Очистка логов
* Очистка аналитики
* Пересоздание системных данных
* Резервное копирование базы данных
* Безопасный сброс CMS

Все критические действия требуют дополнительного подтверждения.
